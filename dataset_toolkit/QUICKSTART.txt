  conda create -n prottex python=3.10 -y
  conda activate prottex
  pip install -r requirements.txt

  ‚Üí Gets 895K samples (1.21 GB)
  ‚Üí Adds PFUD + PSAD data

  python3 scripts/download_mol_instructions_working.py

  ‚Üí Adds more PFUD + PDD + PSAD data

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "datasets library not found"
Fix: pip install datasets huggingface-hub

Problem: "Connection timeout"
Fix: Just run the script again - downloads resume automatically

Problem: "Disk quota exceeded"
Fix: Skip Step 5 (structures) for now, use text-only data
